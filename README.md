# bin-cli-next

本工程是基于bin-ui-next的库模式开发平台，可进行开发自定义组件库并进行预览和调试。

[组件库文档](https://wangbin3162.gitee.io/bin-ui-next/)

## 注意

拉取版本后，全局搜素哦bin-cli-next，并给想要编辑的库更换名字，更换时需注意，部分代码可能需要自行实现样式，如左上角log区等
